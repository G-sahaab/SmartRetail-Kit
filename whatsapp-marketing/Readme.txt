# 💬 WhatsApp Marketing Toolkit

This tool helps business owners quickly send professional marketing messages to their customers.

## 📦 What's Included

- ✅ Pre-written marketing templates: `whatsapp_templates.txt`
- 🧠 Python script to simulate sending messages: `whatsapp_sender.py`

## ⚙️ How to Use

1. Open `whatsapp_sender.py` in any Python environment
2. Enter the customer name
3. Choose message type (1–4)
4. It personalizes and prints the message

✅ Based on real campaigns used at Mir Electronics – Baramulla
